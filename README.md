# Clinic-Patient-Management
This project is a Clinic Patient Management System built with Python. It runs in the terminal and allows clinic staff to manage patient records easily.
The program lets you add new patients, view and search for them, update their information, and record visit notes including the doctor's name, date, and prescription. All data is saved to a JSON file so it stays available the next time you run the program.
It was built using core Python concepts like variables, lists, dictionaries, functions, loops, conditions, file handling, and error handling — all combined into one complete working application.
